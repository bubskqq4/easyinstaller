Features

Multi-Format Support

Install .deb, .snap, .AppImage, .flatpak, .rpm, .tar.gz, .tar.xz, .tgz, .run, and .sh files.

URL Downloads

Download packages directly from URLs with wget.

Kali Aesthetic

Dark theme with neon accents for a hacker vibe.

App Tracking

Track installed apps in a system group and file.
