This Python script helps you quickly switch and apply different themes on your Linux desktop. It automates theme configuration by setting GTK themes, icon packs, and cursor styles using terminal commands.

üìú The Script

import os

# Set your desired theme options
gtk_theme = "Adwaita-dark"
icon_theme = "Papirus-Dark"
cursor_theme = "Breeze_Snow"

# Apply themes using gsettings
os.system(f"gsettings set org.gnome.desktop.interface gtk-theme '{gtk_theme}'")
os.system(f"gsettings set org.gnome.desktop.interface icon-theme '{icon_theme}'")
os.system(f"gsettings set org.gnome.desktop.interface cursor-theme '{cursor_theme}'")

print("‚úÖ Theme applied successfully!")
